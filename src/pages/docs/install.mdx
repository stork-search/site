# Install Stork

<When name="installation-method" value="node">

The easiest way to install Stork is through the `@stork/indexer` NPM module.

Run the following to download the pre-built binary within a module:

```sh
$ yarn add -D @stork/indexer
```

Then, use the module in your project:

```tsx
import stork from "@stork/indexer";

stork.buildIndex({...});
```

</When>

## Pre-built binaries

You can download a pre-built binary to run Stork from the command line. You can either download the binary from the appropriate Github Release or from the Stork CDN.

Pre-built Stork binaries are available for the following platforms:

- macOS (x64, macOS 11)
- macOS (x64, macOS 12)
- macOS (Apple Silicon)
- Linux (Ubuntu 20.04)
- Linux (Ubuntu 22.04)
- Amazon Linux

macOS users can also use Homebrew to install Stork:

```sh
$ brew install stork-search/stork-tap/stork
```

## Build from source

If you have a current Rust toolchain installed, you can use Cargo to build Stork from source.

```sh
$ cargo install stork-search
```

<Callout>
  If these options don't work for you, please [file a Github issue]() describing
  the installation process you hope to see.
</Callout>
